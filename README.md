## Installation

Install with pyenv and poetry:

```bash
git clone https://github.com/killian31/os-cemantix.git
cd os-cemantix
pyenv virtualenv 3.12.2 osc
pyenv activate osc
pip install poetry
poetry install
```

## Launch the app

Run

```python
poetry run python app.py
```

and then open the url in your browser.
